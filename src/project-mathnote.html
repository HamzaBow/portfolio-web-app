Project Overview

Mathnote is a web application for studying and learning mathematics. It is a platform that provides through a simple and intuitive UI the following features:

  • Using many built-in collections of flashcards and cheatsheets.
  • Taking notes (where mathematical formulas are can be easily edited.)
  • Creating custom flashcards.
  • Grouping flashcards and notes in collections.

Features that are coming soon:

  • Quiz editor.
  • Leveraging spaced repetition learning technique to help in studying.
